<h5 id="start"></h5>

### テキスト

<aside>
💡 テンプレート設計では、テキスト要素が一般的に使用されます。以下の属性設定を使用して、テキスト要素の外観や機能を簡単にカスタマイズし、テンプレートにより豊かでプロフェッショナルなデザイン要素を追加できます。
</aside>

<h4 id="text-drag-rotate">
ドラッグ＆回転
</h4>

- テキスト要素をドラッグ：要素ライブラリから【テキスト】要素を選択し、テンプレートの所定の位置にドラッグします。
- 回転とサイズ調整：回転コントロールをクリックして回転させたり、要素の端をドラッグしてサイズを調整したりして、テンプレートのレイアウトに合わせます。

![テキスト_ドラッグ＆回転](../_images/jp/テキスト_ドラッグ＆回転.gif)

#### 属性値設定

#### 1. 基本プロパティ
[`タイトル`](#text-title)：要素に表示される内容を設定します。また、テキストをダブルクリックして内容を編集することもできます。<br/>
[`フィールド名`](#text-field)：要素のフィールド名を定義し、データソースと関連付けます。<br/>
[`モックデータ`](#text-test-data)：モックデータを入力して、実際の効果をプレビューします（先にフィールド名を設定する必要があります）。<br/>
[`位置座標、幅&縦`](#text-position-width-height)：テキストの位置、幅、高さを調整します。<br/>
[`タイトル非表示`](#text-show-title)：テキストタイトルの可視性を制御します（先にフィールド名を設定する必要があります）。<br/>
[`固定位置`](#text-position-width-height-fixed)：固定すると、他の要素の調整に合わせて位置が移動しません。<br/>

#### 2. スタイルプロパティ

[`データ型、フォーマット`](#text-datatype)：データ型と対応するフォーマットを設定します（既存のフォーマットを選択するか、カスタムフォーマットを入力できます）。<br/>
[`フォントスタイル`](#text-fonttype)：テキストのフォント、サイズ、太さ、字間、色をカスタマイズして、デザインのニーズに合わせます。<br/>
[`背景と配置`](#text-backgroundalign)：テキストの背景色、装飾効果、要素内のテキストの配置方法（左右配置、上下配置）、テキストの折り返し、回転角度、要素の階層を設定します。<br/>
[`回転角度`](#text-rotationangle)：要素の回転角度を設定します。<br/>
[`要素の階層`](#text-elementlevel)：要素がレイヤー内で表示される順序を制御します。<br/>

#### 3. 罫線属性
[`罫線のスタイル`](#text-bordertype)：左、上、右、下の罫線のスタイルを個別に設定します。<br/>
[`罫線のサイズと色`](#text-bordercolor)：罫線のサイズと色を調整して、デザインのニーズに合わせます。<br/>
[`内側の余白`](#text-insidemargin)：左、上、右、下の内側の余白を個別に設定します。<br/>

#### 4. 上級プロパティ

`印刷とバーコード設定`：特定の印刷ニーズに対応するために、印刷タイプ、バーコードフォーマット、QRコードの誤り訂正レベルを設定します。<br/>
`表示とドラッグのルール`：テキストの表示ルール、非表示条件、ドラッグ方向を制御します。<br/>
